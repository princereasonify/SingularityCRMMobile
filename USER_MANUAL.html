  1. Tap the + button on the Leads screen.
  2. Fill in: School Name, City, Board, Source, Product Modules, Estimated Value.
  3. The app automatically checks for duplicate leads as you type. A warning banner appears if a match is found.
  4. Tap Add Lead to save.

Moving a Card (Drag & Drop)

  1. Long press a lead card for ~0.4 seconds — the card fades and a ghost card appears under your finger.
  2. Drag the ghost card to the target column. The destination highlights with a "↓ Drop here" indicator.
  3. Release — the lead moves immediately (optimistic update) and syncs to the server in the background.
  4. If the server update fails, the card reverts and an error message is shown.
While dragging, zoom and pan are automatically disabled. A short tap (under 0.4 s) opens the Lead Detail screen as usual.

Priority Filter

The Priority filter ranks schools by a visit priority score based on days since last visit, lead value, and engagement level. Use this to decide which schools to visit first each day.

PRIORITY LEVELSCORE RANGEACTION
🔴 HIGH80–100Visit this week — at risk of going cold
🟡 MEDIUM50–79Visit within 2 weeks
🟢 LOW0–49On track — no urgent action needed

Adding a School — Duplicate Detection

  1. Tap + Add School.
  2. As you type the name and city, the app checks for similar schools in real time.
  3. If matches are found, a yellow warning banner appears with the count.
  4. On submit, a modal shows each match with its type (Definite / Probable / Possible) and a link to view the existing record.
  5. Tap View Existing to inspect, or Create Anyway to proceed.

15. Troubleshooting

ISSUECAUSESOLUTION
Card doesn't move on pipeline drag Long press not held long enough Hold for at least 0.4 seconds before dragging
Red offline banner despite having internet Server health check pending (15s cycle) Wait or go to Settings → Sync Now
AI plan not loading Daily quota exhausted or no internet Check connectivity; quota resets at midnight
Duplicate warning on a unique lead Name/city similarity match Tap Create Anyway to proceed
Login fails with correct credentials Account inactive or network error Check internet; contact your administrator
Background GPS stops (Android) Battery optimisation enabled Settings → Apps → SingularityCRM → Battery → Unrestricted
Background GPS stops (iOS) Location permission not set to Always Settings → Privacy → Location → SingularityCRM → Always
